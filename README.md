# Equipment Leveling Simulator

A simulation tool based on an online game (Black Desert Online).

## 🎯 Goal of the Project
Calculating how many materials in total are used to reach the max level of the equipment.

## ⚙️ How to Run
Requires the [.NET SDK](https://dotnet.microsoft.com/download) (version 9.0 or newer).

```bash
# from the repository root:
cd EquipmentCalculator
dotnet run
```

## 📖 Usage
The program simulates upgrading a piece of equipment from level 0 towards the max level.
For every attempt it prints the current level, the rolled number vs. the chance needed,
and whether it was a success or a fail. At the end it prints the total materials consumed.
